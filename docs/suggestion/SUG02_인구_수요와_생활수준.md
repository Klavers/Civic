# SUG02 — 인구 수요와 생활수준

## 제안 요약

현재 인구층 소비 자원의 충족 여부를 단순 인구 보너스뿐 아니라 `생활수준`으로 집계하고, 생활수준이 과학·세수·인구 수용력에 완만한 보정을 주도록 확장한다.

## 현재 문제

- 인구층 소비는 자원을 소비하지만 플레이어가 왜 특정 소비재를 확보해야 하는지 장기 목적이 약하다.
- 자원 해금이 늘어날수록 소비 항목만 증가하면 기술 발전이 보상보다 부담으로 느껴질 수 있다.
- 인구 수치가 건물 한도에만 사용되면 경제 성장과 시민 생활 사이의 연결이 약하다.

## 권장 구현

- 해금된 인구층 소비 자원별 충족률을 계산한다.
- 필수 수요와 선택 수요를 구분한다.
  - 필수: 식량·기본 의류처럼 미충족 시 생활수준 하락
  - 선택: 가구·기호품처럼 충족 시 추가 보너스
- 생활수준은 0~100 점수로 표시하되, 시뮬레이션에는 구간 효과를 사용한다.
  - 0~39: 불안정
  - 40~69: 안정
  - 70~89: 번영
  - 90~100: 황금기
- 효과는 곱연산 폭주를 막기 위해 작은 가산 보정으로 시작한다. 예: 과학 `+0%/+5%/+10%/+15%`, 세율 효율 `+0%/+2%/+4%/+6%`.

## 플레이 시나리오

고대 진입 후 의류 수요가 해금된다. 의류가 부족하면 생활수준이 `안정`에 머물고 과학 보너스를 받지 못한다. 플레이어가 목축장과 의류 공방의 공급망을 확장하면 `번영`에 도달해 과학과 국고가 증가한다. 이후 기호품이 해금되면 더 높은 생활수준을 노릴 수 있지만, 기존 필수 공급망을 희생하면 점수가 다시 내려간다.

## 기대 효과와 위험

- 소비 자원에 명확한 보상과 목표를 부여한다.
- 생산량 증가 외에 경제 균형을 맞추는 플레이가 생긴다.
- 수요 종류가 너무 많아지면 관리 피로가 커지므로 시대당 신규 필수 수요는 1개 이하를 권장한다.

## 후속 결정 필요

- 필수/선택 수요 데이터 컬럼
- 생활수준 집계식과 자원별 가중치
- 미충족 페널티의 하한
- 생활수준 하락 시 인구 보너스 취소 범위
